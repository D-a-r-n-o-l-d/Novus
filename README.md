# Novus

A script hub for Roblox executors. Built for **Potassium**.

Spiritual successor to Parvus Hub. Generated by AI with human direction. It works.

## Features

- **Aimbot** — Mouse aim, camera snap, prediction, lock aim, auto-fire
- **Triggerbot** — Delay, fire rate, hold click
- **Hitbox Extender**
- **ESP** — Enemy/friendly boxes, health bars, names, tracers, chams
- **FOV Circles** — Visual aim indicators
- **Off-screen Arrows**
- **Radar** — Minimap player tracking
- **Movement** — Walkspeed, jumppower, click teleport
- **Adonis Bypass** — Engine-level, no metatable hooks
- **Keybinds** — Linoria UI, Hold/Toggle/Always modes
- **Config System** — Per-game save/load
- **Unload/Reload** — Clean teardown without rejoining

## Usage

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/D-a-r-n-o-l-d/Novus/master/novus.lua"))()
```

Requires Potassium or a UNC-compatible executor.
